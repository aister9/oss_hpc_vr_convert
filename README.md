# oss_hpc_vr_convert

구글 카드보드를 위한 1인칭 영상의 VR 영상 변환 프로그램

다음과 같은 라이브러리가 사용되었습니다.

* CUDA 11.0
* OpenCV 4.3.0

다음과 같은 언어로 작성되었습니다.

* C++
* C# (WF)

프로그램 실행을 위해서 다음 DLL 파일을 필요로 합니다.
* opencv_world430.dll
* opencv_opencv_videoio_ffmpeg430_64.dll
* oss_vr_converter.dll
* CudaVRConvertDLL.dll

# .netcore runtime 이 설치되어 있어야합니다.
